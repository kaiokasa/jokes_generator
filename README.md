## What is this? 🤔

Joke Generator Studio is a **fun AI toy** that writes jokes for you!Just pick:

- **LEVEL:** easy 😄 / medium 😎 / hard 🤯
- **SETTING:** office 🏢 / school 🎓 / family dinner 🍽️ / online 🌐 / bar 🍺
- **TONE:** sarcastic 🙃 / dry 😐 / wholesome ❤️ / absurd 🤪

…and let AI do the rest! You can even add **extra cues** to spice it up 🌶️.

---

## Quick Start ⚡

1. Clone repo:

```bash
git clone https://github.com/your-username/joke-generator-studio.git
cd joke-generator-studio
```

2. Install requirements:

```bash
pip install -r requirements.txt
```

3. Run interactive demo:

```bash
note2.ipynb
```

* Pick your **LEVEL, SETTING, TONE**
* Add extra cues if you want
* Click **Generate Joke** 🤖🎉

---

## Behind the Scenes 🔧

* **Model:** DistilGPT-2
* **Fine-tuning:** LoRA (lightweight, fast)
* **Data:** Reddit Jokes (filtered & cleaned)
* **Evaluation:** Perplexity + diversity metrics
